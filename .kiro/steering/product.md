# Product Overview

Draftboard is a Canva-like design tool starter kit for creating visual diagrams and designs in the browser. It provides a drag-and-drop canvas interface where users can add shapes, text, images, and manage design elements.

## Core Capabilities

- Canvas workspace with drag-and-drop element manipulation
- Shape library (rectangles, circles) and text elements
- Image upload with optional numeric badges
- Variable system for dynamic text replacement (e.g., `{Force}` â†’ `100N`)
- Properties panel for adjusting colors, sizes, and element attributes
- Layering and rotation controls
- PNG export functionality

## Key Components

- **Canvas Area**: Main 800x600px workspace for design
- **Toolbar**: Left sidebar with element creation tools
- **Image Upload Panel**: Dedicated panel for uploading images with associated numeric values
- **Variable Panel**: Manages reusable variables that can be inserted into text elements
- **Properties Panel**: Right sidebar for editing selected element properties
